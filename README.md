# Future_bot

Бот для ежедневного поиска новых постов во ВК по issue #1.

## Что делает

- При первом запуске создает SQLite-базу `data/future_bot.sqlite3` и заполняет таблицу `База постов ФФ` всеми постами группы `Формула Футурологии`.
- При следующих запусках дополняет `База постов ФФ` постами за последние сутки.
- Каждый цикл собирает посты за последние сутки из `Край Будущего` и `Азимов Учит`.
- Оставляет только посты с ключевым словом `Технология` или хештегом `#Технология`.
- Удаляет посты, ссылки на которые уже встречаются в постах `Формула Футурологии`.
- Перезаписывает таблицу `База новых постов` итоговым списком и отправляет пронумерованные ссылки в чат `2000000170`.

## Настройка

```bash
python -m pip install -e ".[test]"
cp .env.example .env
```

Заполните `.env`:

- `VK_GROUP_TOKEN` - токен группы для чтения стен.
- `VK_USER_TOKEN` - токен пользователя.
- `VK_MESSAGE_TOKEN` - необязательный токен для отправки сообщения, по умолчанию используется `VK_USER_TOKEN`.

## Запуск

Один цикл:

```bash
python -m future_bot --once
```

Ежедневный режим в 03:00:

```bash
python -m future_bot --daemon
```

Время и часовой пояс меняются через `FFBOT_SCHEDULE_TIME` и `FFBOT_TIMEZONE`.
