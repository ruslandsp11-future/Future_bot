# Future_bot

Бот для поиска новых постов во ВК по команде в чате.

## Что делает

- При первом запуске создает SQLite-базу `data/future_bot.sqlite3` и заполняет таблицу `База постов ФФ` всеми постами группы `Формула Футурологии`.
- При следующих запусках дополняет `База постов ФФ` постами, появившимися после последнего сохраненного поста.
- Проверяет чат по явно указанному `peer_id`; стандартный `peer_id` - `2000000015`.
- Раз в 10 секунд ищет команду вида `/поиск по (Технологии, Технология) интервал 5д`.
- Берет слова из скобок как ключевые слова и ищет посты источников за указанный интервал.
- При команде `/поиск по () интервал 5д` берет слова и хэштеги из файла `Список слов и хэштегов.txt`.
- Команда `/стоп поиск` запрашивает остановку текущего поиска, а `/стоп программа` останавливает daemon-режим.
- Перед каждым запуском поиска перечитывает источники из файла `Список групп.txt`; повторные команды в течение 12 часов используют последний опрос групп, если список источников не менялся.
- Если VK не дает прочитать стену отдельной группы, бот пишет ошибку в лог и продолжает поиск по остальным группам.
- Удаляет из `База новых постов` посты, ссылки на которые уже указаны в источниках постов `Формула Футурологии`.
- Отправляет в тот же чат краткий отчет и пронумерованные ссылки от лица группы.
- Для команд из чата показывает прогресс поиска через редактирование сообщения с примерным временем окончания и обновляет процент не чаще раза в минуту; если итоговый список ссылок слишком длинный, разбивает его на несколько сообщений.
- Если VK API возвращает `Flood control` с кодом 9 при отправке или редактировании сообщений, бот делает паузу на 5 минут и повторяет запрос.
- При первом опросе чата отмечает уже видимую историю как прочитанную, чтобы не запускать старые команды после рестарта.

## Настройка

```bash
cp .env.example .env
```

Заполните `.env`:

- `VK_GROUP_TOKEN` - токен группы для проверки чата и отправки сообщений.
- `VK_USER_TOKEN` - токен пользователя для чтения постов групп.
- `VK_MESSAGE_TOKEN` - необязательный токен для отправки сообщений, по умолчанию используется `VK_GROUP_TOKEN`.
- `FFBOT_ALLOWED_USER_IDS` - пользователи, которым разрешено запускать поиск.

Файл `Список групп.txt` содержит группы-источники, по одной группе или ссылке VK на строку.
Файл `Список слов и хэштегов.txt` содержит слова и хэштеги для планового поиска и пустой команды, по одному значению или через запятую.

## Запуск

Для обычного запуска из рабочей директории установка пакета не требуется:

```bash
python -m future_bot --daemon --poll-chat
```

В этом режиме ежедневный запуск по таймеру и команды из чата работают одновременно.

Один цикл:

```bash
python -m future_bot --once
```

Ежедневный режим в 03:00:

```bash
python -m future_bot --daemon
```

Время и часовой пояс меняются через `FFBOT_SCHEDULE_TIME` и `FFBOT_TIMEZONE`.
